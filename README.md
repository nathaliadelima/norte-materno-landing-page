# Método Norte Materno — Página de Vendas

Este projeto é uma landing page desenvolvida para o produto digital **Método Norte Materno**.

O método é voltado para mães, especialmente no pós-parto, que se sentem mentalmente sobrecarregadas, confusas e com dificuldade de organizar a rotina.

A proposta não é produtividade tradicional, mas sim:
- organização consciente
- redução ao essencial
- clareza mental
- regulação emocional
- execução possível dentro da realidade da maternidade

---

## 💻 Tecnologias utilizadas

- HTML5 (estrutura semântica)
- CSS3 (estilização e responsividade)
- JavaScript (interações simples)

---

## 🎯 Objetivo do projeto

Criar uma página de vendas:
- com alta conexão emocional
- narrativa leve e realista
- sem promessas irreais
- com foco em conversão de forma ética

---

## 🎨 Estilo visual

- minimalista
- tons neutros (bege, verde suave)
- foco em legibilidade
- sem poluição visual

---

## 📌 Observações

Este projeto foi desenvolvido com foco em:
- prática de desenvolvimento front-end
- aplicação de conceitos de copywriting
- construção de produtos digitais reais
